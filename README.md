# kotlin-rest-spring
Segue uma simples implementação utilizando:
- kotlin
- Springboot
- gradle

## Clonar o projeto
> git clone [URL_GIT]


## Build and running
Build:
>./gradlew build

Test it:
>./gradlew test

Run it:
>./gradlew bootRun

## Testando a aplicação
> curl http://localhost:8080/greeting?name=Kotlin
